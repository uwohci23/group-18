• As designers we have strived to make the UI as intuitive as possible. However in spite of that it is possible at times users may need assistance while using the system.
  • To accommodate for that we built into the system a landing page for each game so that users can familiarize themselves with the controls and difficulty settings before starting the games.
  • Every game landing page has:
      • Instructions - That are specific to the game and come with audio and visual aids to improve accessibility for all potential users
      Difficulty - So that users can personally alter the game mechanics to their own preference by selecting a difficulty level
