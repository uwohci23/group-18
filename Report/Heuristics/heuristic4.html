The system prevents user error in a variety of ways, consider both mistakes and slips. The allows for better usability, and keeps the application functioning as intended.


User Mistakes:
  • The user cannot exit the game after achieving a high score unless they submit a name for the leaderboard.
User Slips:
  • When exiting games, confirmations are included to avoid slips that would cause the user to lose their progress
  • When modals are open, the rest of the screen is greyed out and is no longer interactable so the user does not accidentally change game state, affect difficulty, etc.
